<h1 id="dast.set.Set">Set</h1>

```python
Set(self, s=set())
```

## &

```python
Set.__and__(self, other)
```

Returns the intersection of self and `other`.

*&* and *intersection* are aliases.

## ==

```python
Set.__eq__(self, other)
```

Returns `True` if self and `other` are equal otherwise `False`.
